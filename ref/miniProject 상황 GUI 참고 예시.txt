선택3 무작정 창문 밖으로 뛰어내린다. x -> 화기는 피했으나 심각한 난상을 입고 말았습니다! 
선택4 벽을 타고 불길이 번지지 않은 옆방으로 탈출을 시도한다. -> x 화기는 피했으나 심각한 낙상을 입고 말았습니다!

E-1
의문의 연기가 도시를 뒤덮었다. 도시에는 실종자가 속출하고 있으며 생존한 사람들이 마트 안에서 구조대를 기다리는 상황, 가족들을 집에 남기고 온 남성은 선택지들 중 한가지 행동을 해야한다.
선택1 입구에 물건들을 쌓아두는 일에만 동참하고 식량은 숨겨둔 채로 아무와도 대화하지 않는다. x -> 남성의 행동이 마트 안의 사람들의 불안한 마음에 악영향을 끼쳤습니다! 불안함을 이기지 못한 정신이상자가 나타가 난동을 부리기 시작했습니다. 
선택2 구조대가 언제 올 것인지 모르므로 식량을 빼앗고자 마트 안의 사람들을 흉기로 협박한다. x -> 남성의 행동이 상황을 악화시켰습니다! 남성은 독박당하였습니다.
선택3 신앙심을 발휘하여 사람들에게 회개하고 종말을 기다리자고 선동한다. x -> 남성의 행동이 상황을 악화시켰습니다! 불안함을 이기지 못한 사람들이 이상증세를 보이기 시작했습니다.
선택4 조용히 상황을 살피고 주위 사람들에게 대화를 시도하며 정보를 수집한다. o -> 사람들은 침착하게 정보들을 공유하기 시작했습니다.

E-2
남성은 연기 속의 괴물이 마을 사람들을 죽이고 있다는 사실을 알게된 상황, 남성은 가족들을 구하기 위해 마트를 나가야 한다. 아래 선택지들 중 한가지 행동을 해야한다.
선택1 흉기를 챙겨들고 쌓아둔 물건들을 치우고 마트 문을 열 것을 협박한다. x -> 남성의 행동이 상황을 악화시켰습니다! 사람들은 공포에 떨기 시작했고 남성은 독박당했습니다.
선택2 신앙심을 발휘하여 사람들에게 회개를 촉구하고 큰소리로 기도한다. x -> 남성의 행동이 상황을 악화시켰습니다! 불안함을 이기지 못한 사람들이 이상증세를 보이기 시작했습니다.
선택3 마트 밖으로 나가기 위해 문을 열어줄 것을 요구한다. x -> 남성의 행동이 마트 안 정신이상자들의 눈에 띄고 말았습니다! 남성은 탈출하지 못했을 뿐 아니라 더욱 탈출하기 어려워졌습니다.
선택4 식량 배분에 앞장서고 마트를 나가려는 사람들을 찾아다니면서 함께 나갈 기회를 찾는다. o -> 남성은 사람들의 신뢰를 얻기 시작했습니다. 마트를 나가려는 사람들은 기회가 생기면 차를 타고 탈출하자는 계획을 세웠습니다. 

E-3
괴물을 피해 마트로 들어온 부상자들에게 지사제가 부족한 상황, 남성은 아래 선택지들 중 한가지 행동을 해야한다.
선택1 지사제를 구하러 나서겠다고 이야기하며 마트 밖의 차를 빌려 줄 사람을 찾는다. o -> 사람들은 공포에 떨면서 어서 지사제를 구해올 것을 요구합니다. 남성은 탈출에 성공했습니다!
선택2 신앙심을 발휘하여 사람들에게 회개를 촉구하고 큰소리로 기도한다. x -> 남성의 행동이 상황을 악화시켰습니다! 불안함을 이기지 못한 사람들이 이상증세를 보이기 시작했습니다.
선택3 두려우므로 흉기를 챙겨들고 무작정 마트 밖으로 나간다. x -> 안개속에서 괴물이 남성을 발견했습니다! 남성은 가족을 찾지 못하고 죽고 말았습니다.
선택4 사람들이 당황한 사이에 혼자서 마트를 탈출한다. x -> 안개속에서 괴물이 남성을 발견했습니다! 남성은 가족을 찾지 못하고 죽고 말았습니다.
